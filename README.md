**MX Diff** is a package for calculating deep differences between two objects.

## Prerequisites

You will need [Node.js](https://nodejs.org/) **version 11+**.

## Instalation

npm:

```
npm install @mx-js/diff
```

yarn:

```
yarn add @mx-js/diff
```
